# 🌱 EcoTrack - 环保碳足迹追踪系统

## 📋 项目概述

EcoTrack是一个基于Vue 3开发的现代化碳足迹追踪应用，帮助用户记录、分析和管理日常生活中的碳排放，通过游戏化机制鼓励环保行为。

### 🎯 核心功能
- **碳足迹记录**：支持交通、饮食、能源、购物四大类别的碳排放计算
- **数据可视化**：实时图表展示碳排放趋势和分类统计
- **挑战系统**：通过环保挑战激励用户持续改进
- **社区互动**：分享环保成果，参与社区讨论
- **用户认证**：完整的登录注册系统，数据隔离保存

## 🛠️ 技术栈

- **前端框架**：Vue 3 + Composition API
- **构建工具**：Vite 5.2
- **状态管理**：Pinia 2.1
- **路由管理**：Vue Router 4.3
- **UI组件库**：Element Plus 2.6
- **样式框架**：Tailwind CSS 3.4
- **图表库**：ECharts 5.5
- **时间处理**：Day.js 1.11

## 🔧 功能详解

### 碳排放计算
支持四大类别的碳排放计算：
- **交通出行**：地铁、公交、私家车、飞机等
- **饮食消费**：牛肉、猪肉、素食、咖啡等
- **能源消耗**：家庭用电、燃气、空调等
- **购物消费**：衣物、电子产品、外卖包装等

### 数据可视化
- **趋势图表**：7天碳排放变化趋势
- **分类饼图**：各类别碳排放占比分析
- **统计卡片**：今日、本周、本月碳排放总量
- **实时更新**：数据变化时图表自动刷新

### 用户系统
- **独立账号**：每个用户的数据完全隔离
- **持久化存储**：基于localStorage的数据保存
- **路由保护**：未登录用户自动跳转登录页
- **状态管理**：完善的用户状态同步机制

### 游戏化元素

```bash

rm package-lock.json
```
