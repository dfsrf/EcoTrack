# 🚀 EcoTrack 项目启动指南

## 📁 项目位置
`C:\Users\Admin\Desktop\新前端`

## 🛠️ 启动步骤

### 1. 打开命令行工具
- 按 `Win + X`，选择 "Windows PowerShell" 或 "命令提示符"
- 或者在 VS Code 中按 `Ctrl + \`` 打开终端

### 2. 进入项目目录
```bash
cd C:\Users\Admin\Desktop\新前端
```

### 3. 安装依赖
```bash
npm install
```

### 4. 启动开发服务器
```bash
npm run dev
```

### 5. 访问应用
启动成功后，在浏览器中访问：
- 本地地址：http://localhost:5173
- 或查看终端显示的实际地址

## 🌟 项目特色

### ✅ 已完成功能
- **Vue 3 + Vite** 现代化开发环境
- **Element Plus** 美观的UI组件库
- **Tailwind CSS** 环保主题样式

## 🛠️ 技术栈

- **前端框架**：Vue 3 + Composition API

## 🚨 故障排除

### 问题1：依赖安装失败
```bash
# 清理缓存并重新安装
npm cache clean --force
rm -rf node_modules
npm install
```

### 问题2：端口占用
```bash
# 使用不同端口启动
npm run dev -- --port 3000
```

### 问题3：Tailwind CSS 样式不生效
- 检查 `vite.config.js` 中的 PostCSS 配置
- 确认 `tailwind.config.js` 路径配置正确
